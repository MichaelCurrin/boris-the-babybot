## Configuration


```bash
$ cd boris/etc
```

Setup config file.

```bash
$ cp config.tmpl.yml config.local.yml
```

Create Tweet messaging content. Add one or more lines to each.

```bash
$ touch hashtags.txt statuses.txt
```
